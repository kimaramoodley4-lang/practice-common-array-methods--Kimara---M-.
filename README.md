# Practice: Using the .length Property

## Project Description
This project is a practice exercise for the **LaunchCode Foundations of Software Development** program. The goal is to manage a local café's order system using JavaScript arrays. 

The system tracks two categories of items: **drinks** and **pastries**. The project demonstrates how to manipulate data dynamically using array properties and methods.

## Key Learning Objectives
* Initializing and managing JavaScript **Arrays**.
* Using the `.length` property to track items dynamically.
* Accessing elements using **Bracket Notation**.
* Implementing **For Loops** to iterate through data.
* Using the `.push()` method to update lists in real-time.

## Technologies Used
* **JavaScript (ES6)**
* **Git/GitHub** for version control.

## How to Run
1. Clone this repository to your local machine.
2. Open the `index.js` file.
3. Run the file using a JavaScript environment (like Node.js) or by linking it to an HTML file and viewing the console in your browser.
